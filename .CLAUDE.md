# Overview Metered Subscriptions Platform
A SaaS billing stack that sells monthly subscriptions with included usage, meters every API call, and hard-stops at quota, with trials, upgrades/downgrades, invoices, and a self-service portal.
This is for API/SaaS vendors (SMB → mid-market) who need simple monthly plans with fixed included units and real-time enforcement—not open-ended overage billing.

## Patterns
**Route handler:**
```javascript
export async function POST(request) {
  const env = getEnv()
  const container = createContainer(env)
  const { logger, call_state, clients } = container.createRequestContext(request.headers)
  // Business logic
  return NextResponse.json(wrapSuccess(data, undefined, call_state.correlationId))
}
```

**Auth:** `const { userId } = await auth()` (Clerk)

**DB:** `db.organization.create()` or `withTx(async (tx) => { ... })`

**Stripe:** `clients.stripe.customers.createOrAttach()` / `clients.stripe.subscriptions.create()`
